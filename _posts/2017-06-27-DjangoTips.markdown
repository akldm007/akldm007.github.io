django-admin-bootstrapped  需要Django 1.6  以上才可以使用

Step:
* django-admin.py startproject xxx
* python manage.py makemigrations
* python manage.py makemigrations
* python manage.py migrate
* python manage.py syncdb
* python manage.py runserver 0.0.0.0:8000
